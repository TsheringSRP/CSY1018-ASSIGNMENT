Learning Process

First of all, I would like to thanks our respective module leader - (lecturer/tutor) Ankit Thapa sir for his dedication in our learning porcess. Ups & Downs

Webpage designing is like an adventure to me. While learning it, I experienced many obtstacles in basic designing process too. A very first problem I faced was div overlapping problem. I had been facing it since the first day of my tutorial class. But I successfully debugged and fixed it in this assignment which later became a great help in my further designing process too. We have criteria to use grid for page layout and better scores. Following the slide (GRID LAYOUT), I began to learn about it and implement it in my assignment. But it wasn't easy as expected. I debugged a lot of overlapping problem here also making me hard to run media query for responsive mobile view design. Similarly, designing hamburger icon was easy but making it appear only mobile phones was really challenging to me. So, after spending much time and giving more effort to it I was finally able to design it according to my concept. And my each issues, solving methods and references are attached in below section of (designing process).
